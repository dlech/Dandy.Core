Remarks
This type is used a a placeholder when a type is needed but no value exists or is needed. For example, since C# can't use System.Void as a type parameter, you can use this instead.
Examples
This shows how Unit can be used where normally System.Void is used. 
class Example
{
    // C# has separate Action and Func<T> classes because Action
    // returns void and Func<T> returns a type. However, using Unit
    // we can make a Func<T> that is equivalent to Action.
    Action func2 = () => doSomething();
    Func<Unit> func1 = () => {
        doSomething();
        return Unit.Value;
    };
}
